# Seed Dataset

The seed dataset in `data/seed_rows.json` contains calibration rows, theorem-backed benchmark rows, and explicitly marked candidates.

The rows cover:

- local discriminant packages for surface singularities;
- the Coble `1/4(1,1)` order-two shadow selector;
- nodal free-relation channels;
- theorem-backed smooth torsion and Enriques-product benchmark rows;
- finite-coefficient cup-product Bockstein rows inspired by the supplied Diaz and Coble-Diaz mechanisms.

The citation metadata is intentionally limited to citation keys supplied in the project brief. Full bibliographic records should be added only when verified.
